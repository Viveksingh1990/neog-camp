    *   Three main ingredients (e.g., `recipeOneIngredient1`, `recipeOneIngredient2`, `recipeOneIngredient3`)

    *   Number of servings (e.g., `recipeOneServings`)

    *   Recipe name
    *   Preparation time
    *   Number of servings
    *   List of ingredients
    *   Description
